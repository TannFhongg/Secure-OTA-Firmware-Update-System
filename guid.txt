# 1. Configure local WiFi and OTA settings
idf.py menuconfig
# -> Example Connection Configuration: set WiFi SSID/password
# -> Secure OTA Example: set HTTPS firmware URL
# -> Secure OTA Example: enable auto OTA on boot only when needed

# 2. Build and flash
idf.py build
idf.py -p COM3 flash monitor

# 3. Build a new firmware version
# Change PROJECT_VER or set a new app version so same-version OTA is skipped.
idf.py build

# 4. Host firmware on a trusted HTTPS server
# The firmware URL must be https:// and the certificate chain must be trusted
# by the ESP x509 certificate bundle or by your custom local CA bundle.
